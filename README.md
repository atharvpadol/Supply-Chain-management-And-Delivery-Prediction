# 📦 Supply Chain Intelligence Dashboard

An end-to-end data analytics and machine learning project designed to analyze supply chain performance and predict delivery delays. This project combines business intelligence with predictive modeling to provide actionable insights into operational inefficiencies.

---

## 🚀 Key Features

- 📊 **Dynamic KPI Dashboard**
  - Late delivery percentage
  - Average profit per order
  - High-risk regions identification

- 📈 **Interactive Visual Analytics**
  - Delay distribution analysis
  - Region-wise delivery risk
  - Profit vs delay impact

- 🔮 **Machine Learning Prediction**
  - Predicts likelihood of late delivery
  - Built using Random Forest Classifier
  - Handles class imbalance using SMOTE

- 🎯 **User-Friendly Web App**
  - Built with Streamlit
  - Clean UI with tabs and interactive charts
  - Real-time prediction interface

---

## 🧠 Business Problem

Delivery delays directly impact customer satisfaction and profitability. This project identifies key drivers of delays and enables proactive decision-making using predictive analytics.

---

## 🛠️ Tech Stack

- Python (Pandas, NumPy)
- Data Visualization (Plotly, Matplotlib, Seaborn)
- Machine Learning (Scikit-learn)
- Deployment (Streamlit)

---

## ⚙️ Machine Learning Workflow

- Data Cleaning & Feature Engineering
- Frequency Encoding for categorical variables
- Handling class imbalance using SMOTE
- Model comparison (Logistic Regression, Decision Tree, Random Forest)
- Final model selection based on Recall (business-critical metric)

---

## 📊 Key Insights

- Delivery delays show minimal seasonal variation → operational issue
- Certain regions have consistently higher delay rates
- Increased delay negatively impacts profitability
- Shipping mode and scheduled shipment days are key predictors
